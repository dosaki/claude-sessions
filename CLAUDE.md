# CLAUDE.md — claude-sessions

## Project Overview

A dashboard tool that displays all running Claude Code sessions. Supports a native GUI window (default, powered by Fyne) and a terminal-based dashboard (`--cli`). Written in Go, cross-platform (macOS, Linux, Windows).

## Build & Run

```bash
make build                                               # build binary
make test                                                # run all tests
make app                                                 # macOS .app bundle (VERSION=dev default)
make app VERSION=1.2.3                                   # macOS .app bundle with specific version
go run ./cmd/claude-sessions --once                      # run without building
./claude-sessions-bin --once                             # quick smoke test
```

Requires Go 1.24+ and CGO (for Fyne's OpenGL bindings). External dependency: `fyne.io/fyne/v2`.

**Dependency graph** (no circular deps):
- `ansi` → nothing
- `config` → nothing
- `discovery` → nothing
- `focus` → `discovery`
- `gui` → `config`, `discovery`, `focus`
- `terminal` → `ansi` (for CursorShow in SIGTERM handler)
- `render` → `ansi`, `config`, `discovery`, `terminal`
- `cmd/main` → `ansi`, `config`, `discovery`, `focus`, `gui`, `render`, `terminal`

## Constraints

- External dependency: Fyne (`fyne.io/fyne/v2`) for GUI mode. CGO required
- Cross-platform: macOS, Linux, Windows (platform-specific code uses build tags)
- ANSI escape sequences must be handled when measuring/truncating strings — use `ansi.VisibleLen()` and `ansi.Truncate()`
- CPU is checked first for status; JSONL state is the secondary signal (see `discovery.DetermineStatus`)
- Windows limitations: CPU% is unavailable (status relies on JSONL); CWD detection is best-effort; session focus is IDE-only

## Common Tasks

- **Add IDE detection**: add entry to `idePatterns` in `internal/discovery/filter.go`, or update parent-process switch in `platform_unix.go`/`platform_windows.go`
- **Add column**: update `Session` struct in `classify.go` → populate in `ClassifySessions()` → adjust CLI rendering in `render.go` → add GUI column in `gui.go`
- **Add sort column**: add to `ValidSortColumns` and `SortSessions` switch in `internal/discovery/sort.go` → add to `columnSortKeys` in `gui.go`
- **Add focus support for new terminal**: add to `terminalPatterns` in `internal/focus/focus_unix.go` → add case in `FocusSession()` if special logic needed
- **Change status logic**: modify `DetermineStatus()` in `internal/discovery/classify.go`
- **Modify GUI theme**: edit `internal/gui/theme.go`
- **Add ANSI helpers**: add to `internal/ansi/ansi.go`
- **Add terminal source**: update parent-process switch in `platform_unix.go` and/or `platform_windows.go`
